# andor.cool
Source code of my visual journal
<http://andor.cool>

## Requirements
- Linux, or macOS
- NodeJS 14.15.1 or later

## Installation

    git clone https://github.com/hngrhorace/andor.cool.git
    cd andor.cool
    git submodule update --init --recursive --remote
    npm install
    npm run build
